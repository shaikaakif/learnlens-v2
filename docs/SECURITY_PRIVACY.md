# Security and Privacy

Student educational data is sensitive.

Use:
- role-based access,
- Row Level Security,
- encryption,
- server-side secrets,
- data minimization,
- controlled retention.

Default MVP: temporary answer-sheet processing rather than permanent raw-image storage.

Question papers/rubrics may be privately retained because they are reused.

Before production, verify the current AI provider's API data-processing and retention terms. Do not claim zero retention unless verified.

The What Next? export should omit unnecessary names, IDs, school names, and personal identifiers.

Production deployment involving minors requires appropriate institutional/parental governance and legal review.
