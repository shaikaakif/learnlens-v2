# User Flows

## Student — School Mode
Login → Dashboard already knows student/class → Select available assessment → Scan/upload answer-sheet pages → Confirm pages → Analyze → Learning MRI → Evidence → What Next? → Generate/copy tutor context → Open preferred AI tutor → Return for future assessment → Compare progress.

## Student — Personal Mode
Open LearnLens → Preferred name → Grade/class → Optional age → Subject → Upload question paper → Upload answer sheet → Optional marking scheme → Analyze → Learning MRI.

## Teacher
Login → Select class → Create assessment → Upload question paper → Optional rubric → AI extracts questions → Verify → Publish → View class diagnostic dashboard → Inspect patterns → Validate findings.

## Parent
Login → Linked child → Latest assessment → Strengths → Primary improvement opportunity → Trend → Recommended support.

## Admin
Manage users/classes → permissions → branding → retention policy → usage/integration settings.
