# Pravinya MVP Scope

## Goal
Demonstrate the core LearnLens experience, not a production-ready company.

## Must Work
- Professional responsive UI
- Student demo account/profile
- Assessment selection
- One preconfigured assessment
- Answer-sheet upload
- Secure Gemini multimodal API call
- Structured analysis
- Learning MRI
- What Next? context generation
- Copy context
- Open preferred external AI assistant
- Basic progress/history view

## Recommended
- Supabase Auth/database
- Teacher dashboard with seeded data
- Parent view with seeded data
- Assessment creation UI
- Question-paper upload
- Azure Cosmos DB

## Can Be Seeded
- Large class analytics
- Months of historical data
- ML trend examples
- ERP integration
- Billing

## Not Required
- Custom foundation model
- Perfect OCR
- Native mobile app
- Full ERP
- Production billing
- Autonomous official grading

## Golden Demo
31/40 → Select English assessment → Upload corrected sheet → Gemini analysis → Learning MRI → Explain root cause → What Next? → Teacher class insight → Parent simplified insight.
