# UI/UX Direction

## Brand
LearnLens AI — See Beyond the Score.

## Visual Style
Professional, minimal, calm, modern EdTech.

Palette direction:
- light pistachio green,
- white,
- subtle soft gradients,
- dark readable typography,
- restrained status accents.

Avoid crowded dashboards, excessive icons, childish gamification, and neon “AI” aesthetics.

## Learning MRI Layout
1. Snapshot
2. Strengths
3. Where Marks Were Lost
4. Evidence
5. Recurring Patterns
6. What Next?

## Confidence UX
Use labels such as High confidence, Needs review, Unable to determine.

Never visually disguise uncertain AI interpretation as fact.
