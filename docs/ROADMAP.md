# Roadmap

## Phase 0 — Competition
Documentation → design system → student golden path → Gemini integration → Learning MRI → seeded teacher/parent views → demo testing.

## Phase 1 — Pilot
One school/class/subject → teacher validation → real feedback → measure usefulness and accuracy.

## Phase 2 — Diagnostic Quality
Better question matching → handwriting confidence → subject-specific taxonomies → rubric grounding.

## Phase 3 — Longitudinal Intelligence
Trend engine → recurring-error detection → intervention tracking → personalized learning profile.

## Phase 4 — School Platform
SSO → ERP/LMS integration → white-label → school groups.

## Phase 5 — ML
Only with sufficient validated and appropriately governed data: pattern models and intervention-effectiveness analysis.
